# 三菱重工業（7011）投資分析レポート

## ▶ 投資メリット・デメリット概要

### ✅ メリット

| 項目 | 内容 |
|------|------|
| **防衛需要の急拡大** | 日本の防衛費がGDP比2%に向け急増。防衛受注は2023年度で1兆6,803億円と国内トップクラス。2027〜2029年度には防衛事業で年1兆円超の売上を計画 |
| **ガスタービン独占的優位** | 世界的なデータセンター・AI需要増によるガスタービン需要が急拡大。2024年の年間発注量は57.4GWと拡大続く。長期アフターサービス契約も収益を安定化 |
| **アナリスト強気評価** | 2026年5月時点でアナリスト15人中、強気買い10人・買い2人・中立2人。平均目標株価5,374円（現在比約+29%） |
| **中期計画の進捗良好** | 2024事計の最終年度（2026年度）目標：売上高5.7兆円・事業利益4,500億円。受注高は6.7兆円に上振れ見込み |
| **宇宙・航空の長期成長** | H3ロケットの商業化が進み、欧州衛星企業との受注獲得も開始。GCAPで次世代戦闘機の主要産業パートナー |

### ⚠️ デメリット・リスク

| 項目 | 内容 |
|------|------|
| **地政学リスク依存** | 防衛・ガスタービン需要は地政学情勢に左右されやすく、緊張緩和局面では株価の調整要因になりうる |
| **為替リスク** | 海外売上比率が高く、円高転換は業績・株価に下押し圧力。輸出依存度が高い事業構造 |
| **プロジェクトリスク** | 大型長期プロジェクト（GCAP・宇宙等）は遅延・コスト超過リスクを内包。過去にスペースジェットで巨額損失を計上した前例あり |
| **バリュエーション割高感** | 株価急騰後のPERは業界平均より高く、好材料の一定程度織り込み済みとの見方も |
| **人材・サプライチェーン** | 生産能力拡大に伴う人材不足・サプライチェーン逼迫リスク。横浜・名古屋の工場拡張も進行中 |
